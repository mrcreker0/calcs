# 📘 Конспекты лекций по математическому анализу

Этот репозиторий содержит конспекты лекций по курсу **математического анализа** (матан).

## 📂 Содержимое

* `.tex` файлы с оформленными лекциями
* PDF-версии для удобного чтения
* вспомогательные файлы конфигурации (`config.tex`, шаблоны и т.д.)

## ⚙️ Как использовать

1. Установите LaTeX (например, TeX Live или MiKTeX)
2. Скомпилируйте основной файл:

```bash
pdflatex main.tex
```

или используйте любой LaTeX-редактор (Overleaf, TeXstudio и т.п.)
